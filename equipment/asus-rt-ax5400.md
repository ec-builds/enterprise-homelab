# ASUS RT-AX5400 — Spec/Reference Sheet

Dual-band WiFi 6 (802.11ax) extendable router.

## Wireless

| Item | Spec |
|---|---|
| Standard | WiFi 6 (802.11ax); backward compatible with 802.11ac/n/g/b/a |
| Class | AX5400 (574 + 4804 Mbps) |
| 5 GHz | Up to 4804 Mbps (WiFi 6) |
| 2.4 GHz | Up to 574 Mbps (WiFi 6) |
| Bands | 2.4 GHz / 5 GHz |
| Bandwidth | 20/40/80/160 MHz |
| Speed tech | OFDMA, MU-MIMO, 1024-QAM, standard-based & universal beamforming |
| Antennas | 4 external, non-detachable |

## Hardware

| Item | Spec |
|---|---|
| CPU | 1.5 GHz tri-core |
| RAM | 512 MB |
| Flash | 256 MB |
| WAN | 1× Gigabit RJ45 |
| LAN | 4× Gigabit RJ45 |
| USB | 1× USB 3.2 Gen 1 |
| Buttons | WPS, Reset, Power |
| LEDs | Power, 2.4G, 5G, LAN ×4, WAN |
| Power | AC 110–240 V in; DC 19 V / 1.75 A |
| Dimensions | 161.05 × 226.23 × 160.23 mm |
| Weight | 612.4 g |

## Operating Modes

Wireless router, access point, media bridge, repeater, AiMesh node.

## Key Features

- **AiMesh** — works as primary router or mesh node
- **AiProtection Pro** (Trend Micro) — malicious site blocking, two-way IPS, infected device blocking, security assessment; lifetime free
- **VPN** — client & server: OpenVPN, WireGuard, PPTP, L2TP (client), IPSec (server); VPN Fusion; Instant Guard
- **Security** — WPA3-Personal, WPA2/WPA-Personal & Enterprise, Let's Encrypt, DNS-over-TLS, SSH, firewall (keyword/URL filters ×64, service filters ×32)
- **Parental controls** — site/app blocking, safe browsing, internet schedules
- **Guest network** — 3× per band, time limits, WPA3 support
- **QoS/traffic** — Adaptive QoS, bandwidth limiter (32 rules), traffic monitor & analyzer with website history
- **WAN** — PPPoE/PPTP/L2TP/auto/static IP, Dual WAN, WAN aggregation, 3G/4G LTE dongle, port forwarding (64), port triggering (32), DMZ, DDNS
- **LAN** — DHCP, IPTV, LAN link aggregation, manual IP assignment (64), Wake-on-LAN
- **USB apps** — Samba/FTP server, media server, Time Machine, AiCloud, Download Master, AiDisk; HFS+, NTFS, vFAT, ext2/3/4

## Package Contents

Router, RJ-45 cable, power adapter, quick start guide, warranty card.
