# Cisco WS-C3560CG-8PC-S — Spec/Reference Sheet

Catalyst 3560-CG compact series: fanless, managed Gigabit PoE+ switch with IP Base, for deployments outside the wiring closet (retail, classrooms, offices).

## Ports

| Item | Spec |
|---|---|
| Downlinks | 8× 10/100/1000 RJ45, PoE+ |
| Uplinks | 2× dual-purpose: 1G copper RJ45 or 1G SFP each |
| Console | RJ45 console (accessory kit includes console cable) |

## PoE

| Item | Spec |
|---|---|
| Standards | IEEE 802.3af (PoE), 802.3at (PoE+) |
| Per port | Up to 30 W |
| Total budget | 124 W |

## Performance

| Item | Spec |
|---|---|
| Switching capacity | 32 Gbps |
| Forwarding rate | 14.9 Mpps (64-byte packets) |
| Jumbo frames | 9,018 bytes |
| VLANs | 1,005 active; 4K VLAN IDs |

## Hardware

| Item | Spec |
|---|---|
| DRAM | 128 MB |
| Flash | 64 MB |
| Cooling | Fanless (silent) |
| Power | Internal PSU, 100–240 VAC, 1.7–0.8 A |
| Dimensions | 1.75 × 10.6 × 9.4 in (1RU; desk, wall, or rack mount w/ kit) |
| Operating temp | 23–113 °F (-5–45 °C) |
| Humidity | 5–95% non-condensing |
| MTBF | ~355,830 hours |

## Software / Features

- **IOS IP Base** — L2 switching plus basic L3: static routing, RIP, OSPF
- **Security** — 802.1X, MACsec, Cisco TrustSec, DHCP snooping, dynamic ARP inspection, port security, ACLs
- **Management** — CLI (console/SSH), SNMP, Cisco Network Assistant; supports QoS, voice VLAN, Auto Smartports
- **No stacking**, no redundant/hot-swap PSU or modules

## Typical Uses

IP phones, wireless APs, IP cameras at the network edge; conference rooms, POS/retail counters, classrooms — anywhere quiet, compact PoE switching is needed.

## Notes

- End-of-sale/end-of-life product line (superseded by Catalyst 3560-CX); commonly available refurbished (~$250–300).
- Rack mounting requires separate compact-switch rack mount kit (RCKMNT-19-CMPCT).
