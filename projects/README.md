# 🗂️ Projects

Hands-on projects that make up the **enterprise-homelab** environment. Each subfolder contains its own documentation, configurations, and lessons learned.

## Documentation Standard

Each project folder should include:

1. **README.md** — overview, objectives, and architecture
2. **Configuration files / scripts** — sanitized of secrets
3. **Documentation** — setup guides, troubleshooting notes, lessons learned
