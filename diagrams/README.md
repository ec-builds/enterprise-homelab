# 📐 Diagrams

Architecture and topology diagrams for the **enterprise-homelab** environment.

## Purpose

This folder contains visual documentation of the lab's design, including:

- **Network topology** — physical and logical layouts, VLAN segmentation, firewall zones
- **Infrastructure architecture** — hypervisors, virtual machines, containers, and storage
- **Service dependency maps** — how applications and services interact
- **Identity & access flows** — Active Directory / Entra ID hybrid identity diagrams
- **Backup & DR workflows** — data protection and recovery paths

## Conventions

- Export diagrams as `.png` or `.svg` for embedding in documentation
- Include the editable source file when possible (e.g., `.drawio`, `.vsdx`) so diagrams can be updated later
- Use versioned or dated filenames for major revisions (e.g., `network-topology-2026-06.png`)

## Recommended Tools

- [draw.io / diagrams.net](https://www.diagrams.net/) — free, with versionable XML source files
- [Lucidchart](https://www.lucidchart.com/)
- [Excalidraw](https://excalidraw.com/) — great for quick sketches
