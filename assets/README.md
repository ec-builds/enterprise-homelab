# 🎨 Assets

Branding and visual assets for the **enterprise-homelab** repository.

## Contents

| File | Description |
|------|-------------|
| `enterprise-homelab-logo.png` | Official project logo used in the main README and documentation |
| `roadmap.png` | Visual roadmap outlining planned phases and future milestones for the lab |

## Usage

Reference these assets in documentation using relative paths:

```markdown
![Enterprise Homelab Logo](../assets/enterprise-homelab-logo.png)
```

## Guidelines

- Keep image file sizes optimized for fast page loads (prefer PNG for logos/diagrams, compress before committing)
- Use descriptive, kebab-case filenames (e.g., `network-topology-v2.png`)
- Update this README when adding new assets
